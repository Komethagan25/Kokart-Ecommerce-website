# KO Kart 🛒-Ecommerce-website


A full-stack e-commerce web application built with React, Node.js, MongoDB, and Firebase.

🌐 **Live Demo:** https://kokart-ecommerce-website-1gra.vercel.app


## Features

### User Features
- 🔐 Authentication (Email/Password + Google Sign-In)
- 🏠 Home page with banner slider and product grid
- 🔍 Search products with suggestions
- 📦 Product details page with image swiper
- 🛒 Add to cart, increase/decrease quantity
- 💳 Razorpay payment integration
- 📍 Address management at checkout
- 📋 Order history with live tracking progress bar
- ⭐ Rating and review system
- ❤️ Wishlist (save products)
- 🏷️ Category filter (Electronics, Clothing, etc.)
- 👤 Profile page with saved addresses

### Admin Features
- 📊 Dashboard with stats (orders, revenue, users, products)
- 📈 Revenue chart (last 7 days)
- ➕ Add products with multiple images
- 🗑️ Delete products
- 📦 View and manage all orders
- 🔄 Update order status (Placed → Confirmed → Shipped → Out for Delivery → Delivered)


## Tech Stack

### Frontend
| Technology | Purpose |
|---|---|
| React | UI framework |
| Tailwind CSS | Styling |
| React Router | Navigation |
| Axios | API calls |
| Firebase Auth | Authentication |
| Swiper.js | Image slider / Banner |
| Recharts | Admin dashboard charts |
| Razorpay | Payment gateway |

### Backend
| Technology | Purpose |
|---|---|
| Node.js | Runtime |
| Express.js | Web framework |
| MongoDB | Database |
| Mongoose | ODM |
| Firebase Admin | Token verification |
| Cloudinary | Image storage |
| Multer | File upload |
| JWT | Authorization |


### Clone the repo
```bash
git clone https://github.com/yourusername/kokart.git
```

### Setup Backend
```bash
cd backend
npm install
```
Run backend:
```bash
node server.js
```

### Setup Frontend
```bash
cd frontend
npm install
npm run dev
```

## Deployment

- **Frontend** → Vercel
- **Backend** → Render
- **Database** → MongoDB Atlas
- **Images** → Cloudinary
